# POKÉBATTLES
POKEBATTLES is a retro-inspired Pokémon battle simulator built entirely with HTML, CSS, and JavaScript. Designed with a pixel-art aesthetic and modern visual effects, the game delivers an engaging turn-based battle experience directly in the browser without requiring any external frameworks. Features 🎮 Classic turn-based Pokémon battle system .
