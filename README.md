# POKÉBATTLES

POKEBATTLES is a retro-inspired Pokémon battle simulator built entirely with HTML, CSS, and JavaScript.

---

## ✨ FEATURES

- 🎮 Classic turn-based Pokémon battle system
- ⚡ Dynamic attack animations for multiple Pokémon types
- ❤️ Real-time HP bars and damage indicators
- 🌿 Type effectiveness calculations
- 🔥 Custom visual effects
- 🎨 Retro pixel-art interface
- 🌌 Animated backgrounds and particles
- 👥 Party selection and Pokémon switching
- 📱 Responsive design
- 🚀 Lightweight and framework-free

---

## 🎮 GAMEPLAY

Choose your Pokémon team, enter the battle arena, and challenge opponents using strategic move selections.

## 🛠️ TECHNOLOGIES

- HTML5
- CSS3
- JavaScript

## 🌟 VISION

POKEBATTLES aims to recreate the excitement of classic Pokémon battles in the browser.
